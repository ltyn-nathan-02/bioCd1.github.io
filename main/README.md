# Nathan Liu — Personal Site

Built with [Next.js](https://nextjs.org) (App Router) and hosted on **GitHub Pages**.

- Node graph navigation (draggable nodes)
- Starfield + meteor background animation
- Static export (deploys via GitHub Actions)

### Dev

```bash
cd main
npm install
npm run dev
```

### Deploy

Push to `main` on GitHub — CI builds and deploys to `bioCd1.github.io` automatically.
